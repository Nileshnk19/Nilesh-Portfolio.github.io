Projects

NatureWalk App

An iOS application for a recreation facility that promotes guided group nature walk sessions in a city. The app allows users to login, view available sessions, select a session to view details, share session information, add sessions to their favorites list, purchase tickets, view purchased tickets, and maintain their profile. The app also has Cloud Firestore for data persistence, FirebaseAuth for user authentication, and implement a login screen with a Remember Me feature. Along with that it also consists a user-friendly interface with bottom tab views and toolbar items for easy navigation.

Online Video Tutorial App

A mobile app for an online education company that provides a video tutorial series for learners. The displays a list of video lessons, allow users to view a video, and track which lessons have been completed. The app consists of four screens: Enter Your Name, Welcome Back, Lessons List, and Lesson Details. The navigation flow starts with the Enter Your Name screen, where users enter their name and proceed to the Lessons List screen. The Lessons List screen displays a RecyclerView of lessons, and the Lesson Details screen shows detailed information about a specific lesson, including a link to the lesson. Users can mark a lesson as completed and the app will remember their progress using SharedPreferences. Data storage is implemented with SharedPreferences and the (key,value) pairs should be designed for easy data retrieval.

Property Rental App

Embark on a digital journey with two Android apps, Landlord and Renter, for renting properties. The Landlord app allows property owners to create and manage listings, while the Renter app enables users to search and filter properties, add to watchlists, and view details. Both apps utilize Firebase Firestore for data storage and Firebase Authentication for user login and logout operations. .
